# 데이터 기반 보고서 생성 자동화

**Python·SQL 기반 성과 보고서 자동화 파이프라인 구축**

---

## 프로젝트 개요
- 수작업 중심으로 운영되던 성과 보고서 작성 프로세스를 분석하고, 자동화 필요 요소 도출  
- 대상 시기와 타겟이 매번 달라지면서 반복적으로 수정해야 했던 데이터 추출과 지표 산출 구조 개선  
- 보고서 작성 효율과 일관성을 높이기 위해 SQL 기반 자동화 시스템 설계 및 구현

---

## 구현 내용
- SQL 기반 데이터 추출 구조 설계 및 모듈화  
- 데이터 가공, 시각화, 리포트 생성으로 이어지는 자동화 파이프라인 구축  
- 오가닉 지표, 팔로워 수 등 주요 성과 데이터와 키워드 분석 결과를 자동 반영  
- 반복 업무 시간을 단축하고 분석 기준의 일관성을 확보하도록 구조화

---

## 파이프라인 흐름
1. **데이터 추출**: 광고, 캠페인, 콘텐츠 관련 데이터 수집  
2. **데이터 가공**: 타겟별, 기간별 분석 지표 산출 및 집계  
3. **키워드 분석**: SQL 내에서 키워드 및 키워드 조합 분석  
4. **리포트 생성**: 자동 보고서 작성 및 시각화  
5. **자동화 실행**: 파라미터 입력만으로 반복 업무 실행

---

## 주요 성과
- 수작업 중심 보고서 작성 구조를 파라미터 기반 자동 실행 구조로 전환  
- 반복 업무 시간을 대폭 단축  
- 분석 결과 재현 가능하도록 구조화, 실무 의사결정 활용도 향상

---

## 기술 스택
- Python, SQL  
- 데이터 분석 및 시각화: Pandas, Matplotlib/Seaborn  
- 자동화 스크립트 기반 보고서 생성
